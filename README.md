# Task 1: Understanding Dataset and Data Types

## 📌 Objective
To explore and understand the dataset using Exploratory Data Analysis (EDA) techniques.

## 🛠 Tools Used
- Python
- Pandas
- NumPy

## 📂 Dataset
- CSV dataset containing structured data

## 🔍 Steps Performed
- Loaded dataset using Pandas
- Displayed first few rows using head()
- Checked dataset information using info()
- Generated statistical summary using describe()
- Identified missing values using isnull()
- Analyzed data types of each column
- Differentiated numerical and categorical features

## 📊 Key Observations
- Dataset contains both numerical and categorical data
- Some columns had missing values
- Data types were identified for further processing

## ✅ Outcome
Successfully understood dataset structure, data types, and basic statistics which helps in further data preprocessing.
